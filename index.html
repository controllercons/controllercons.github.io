How to use

Webfont

Linking Controllercons

How Controllercons are used has changed a little in 2.0. You now need to add the Controllercon class name to reference a specific controller alongside an extra class to control its style.

<!-- Solid Xbox Series X -->
<i class="ccs cc-xbox-series-x"></i>

<!-- Outline PlayStation 5 -->
<i class="cco cc-ps5"></i>

<!-- Duotone Switch Joy-Cons Gripped -->
<i class="ccd cc-joycons"></i>

Classes can be found by clicking on any of the Controllercons from the grid above or by looking on the demo link below.

Style classes come in three flavours:

  • .ccs (for solid)
  • .cco (for outline) &
  • .ccd (for duotone).

To get up and running you need to download the webkit.zip file and extract the controllercons folder into your project.

Then add a link to the main controllercons.css file

<link rel="stylesheet" type="text/css" href="/controllercons/controllercons.css">

Now add a link to one, or more, of the Controllercon style stylesheets

<link rel="stylesheet" type="text/css" href="/controllercons/controllercons-solid.css">

If you'd like to use all of the styles you can alternatively link the controllercons-all.css stylesheet

<link rel="stylesheet" type="text/css" href="/controllercons/controllercons-all.css">

Styling Controllercons

Controllercons will inherit their colours by default but you can override these by adding a css variable reference to the root of your stylesheet

:root {
	--controllercon-colour-1: #bada55
}

Duotone Controllercons are slightly different and make use of a couple of extra variables

:root {
	--controllercon-colour-2: inherit;
	--controllercon-opacity: 0.5;
}

You can adjust these variables to completely tailor the look of your duotone Controllercons.

Just make sure that your main stylesheet is loading after the Controllercon stylesheets!

Upgrading from 1.x to 2.0

Where possible I'd recommend changing your mark-up to match the new way to integrate Controllercons but I appreciate that isn't always possible.

In cases where you aren't able to do this, you can link the controllercons-legacy.css stylesheet and all your old classes will work.

<link rel="stylesheet" type="text/css" href="/controllercons/controllercons-legacy.css">

Typekit

Download the typekit.zip file and extract it to your device.

Install either the .otf (recommended) or .ttf font files.
How to: (Mac OS | Windows)

Once installed, you'll see the Controllercon font family appear in your software of choice. The solid, outline and duotone styles can be selected from the style dropdown where you'd usually find thin, regular & bold weights.

Controllercon glyphs can be found for each of the icons from the grid above or from the demo.psd file included in the typekit.zip file.
